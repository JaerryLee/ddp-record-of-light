# Zone 3 · 살림터 (Salimter)

> DDP 옥상. 서울 야경을 파노라마로 담고, 시간제 퍼즐과 엔딩을 진행.
> 소요 시간 목표: **6~9분** · 난이도: ★★~★★★ (C6는 시간 압박)

## 1. 공간 레이아웃

```
   [ Z2 Elevator Exit ]
          │
          ▼
   ┌──────────────────────────────┐
   │   잔디 언덕 (Rooftop Hill)    │  ← C5 퍼즐
   │        ~~ ~~ ~~               │
   │   [관측 데크]                 │  ← 서울 파노라마 스폿
   └──────────────┬───────────────┘
                  │
                  ▼
   [ 끝단 원형 플랫폼 ]         ← C6 (엔딩 전용)
                  │
                  ▼
        [ Ending Sequence ]
```

면적 ≈ 70 m × 40 m. 개방 공간. 외곽은 유리 난간+역장으로 낙하 불가.

## 2. 진입 시퀀스 — 엘리베이터 독백

엘리베이터 내부 1인칭. 40초. 창밖으로 DDP 곡면이 지나간다.

```
AI: "율. 질문 하나 해도 될까요."
AI: "당신이 가장 오래된 기억이라고 부르는 것은, 정말 당신의 기억인가요."
AI: "… 아니요. 대답 안 하셔도 돼요. 저도 답이 없어요."
(침묵 12초)
AI: "옥상이에요. 바람이 셀 거예요."
```

문이 열리면 쿨 블루-바이올렛 그라데이션의 서울 밤하늘이 정면에 펼쳐진다. BGM이 교체되며 3초간 모든 SFX 덕킹.

## 3. 퍼즐 P3.1 — "옥상의 두 사람" (C5)

### 목표
Splitter + Filter(G) + Mirror로 Green 빔을 잔디 언덕 너머 C5에 전달.

### 공간 배치
```
  광원 F ─► [Splitter] ─G─► [Mirror]  ┐
                                      │
                  (언덕 위로 반사)     ▼
                                   [Filter G] ─► [ C5 ]
```

### 설계 포인트
- 잔디 언덕의 곡선을 따라 빔이 "흘러가는" 느낌 연출
- 플레이어가 언덕 위에 올라가 빔 경로를 위에서 내려다볼 수 있는 시점 제공
- 이동하며 **파노라마에 자연스럽게 노출**되도록 Node 배치

### Vignette (C5 · 옥상의 두 사람, 25초)
- 2041년 여름. 20대 커플. 바람 소리 강.
- 여자: `"여기 옥상, 서울 제일 좋은 곳. 너만 아는 거 해줘."`
- 남자: `"음… 그럼, 여기서 지금 이 바람 냄새. 나중에 내가 까먹으면, 네가 나한테 알려줘."`
- 여자: `"응. 약속."`
- **여자의 목소리 음색이 아카이브 AI와 명백히 일치**. 플레이어가 알아채도록 1초간 AI 라인 끊기

## 4. 퍼즐 P3.2 — "대정전의 밤" (C6, 시간제)

### 목표
Mirror+ ×2 사용. Mirror+는 **30초 주기로 스스로 45° 회전**. 두 Mirror+의 회전 위상을 읽어 빔이 C6에 도달하는 "윈도우"를 포착해 프리즘을 배치.

### 공간 배치
```
  광원 G ─► [Mirror+ A] ─► ... ─► [Mirror+ B] ─► [ C6 ]
              (30s 주기)            (30s 주기, 15s 위상차)
```

### 설계 의도
- 게임 엔딩 직전 **긴장감**. 다만 실패 페널티 없음 → 다시 풀면 됨
- 시간제이지만 플레이어가 관찰→행동의 리듬을 갖도록 30초 사이클은 충분히 긴 편

### 힌트
- 광원 G 옆 안내판 음각으로 `"기다릴 줄 아는 빛이 있다"` 문구
- 3분 이상 정체 시 AI: `"빛도 숨을 쉬어요. 들숨에 놓는 게 아니라, 날숨에 놓는 거예요."`

### Vignette (C6 · 대정전의 밤, 45초, 엔딩 프리앰블)
- 2042년 3월 23일 저녁. 관리자 긴급 로그
- 목소리: 차분하게 시작 → 로그 후반 긴장 → 마지막 체념
- 마지막 문장: `"… 백업이 끊겼어요. 이건 마지막으로 저장합니다. 유선희."` (AI 본인 이름)
- 활성화 즉시 엔딩 시퀀스로 전환

## 5. 엔딩 시퀀스 (Non-Interactive, 180초)

```
T+0s    C6 활성화. 전 존의 조명이 도미노로 점등
T+20s   플레이어 전방 파노라마 자동 팬 (1인칭 유지, 마우스 룩 잠금)
T+40s   아카이브 AI 엔딩 내레이션 시작
T+90s   파노라마 넓어지며 DDP 외벽 실루엣이 화면 가장자리에 들어옴
T+130s  서울 야경의 등이 하나하나 들어오는 타임랩스 (압축 30초)
T+160s  블랙 아웃, 사운드 페이드
T+170s  "2045년 3월 24일, 새벽 5시 30분" 타이틀 카드
T+180s  크레딧 롤 시작 (3분)
```

### 엔딩 내레이션 요지 (전문은 02 · Narrative에서 관리)
- "고마워요, 율." — 아카이브의 자아 인식 순간
- "도시는 누군가의 하루 없이는 도시가 아니에요"
- "이제 당신의 하루가 여기에 저장됐어요"

### 선택 인터랙션
- 엔딩 중 단 한 번, 10초 창에 `E` 누르면 자막에 율의 한 줄이 나타남: `"… 고맙습니다."`
- 누르지 않아도 엔딩 동일. **선택이 엔딩을 바꾸지 않는 것 자체가 주제** (기억은 당사자의 선택으로만 존재)

## 6. 환경 내러티브

- 잔디 언덕 위 **픽셀 불빛**: 수십 개의 꺼진 "기억 등" 조형물. 엔딩에서 점등
- 관측 데크 난간에 플레이어 이름 각인 자리 (2차 확장용, 1차 빈 자리로 둠)
- 바람에 흩날리는 **낙엽 파티클** — 시기와 무관하게 서정 연출

## 7. 라이팅

- **Key**: 달 (쿨 화이트 #E8EEF5, 강도 0.6)
- **Fill**: 서울 야경 큐브맵 reflection + 약한 사이언 rim
- **Accent**: 엔딩에서 하나씩 켜지는 앰버 포인트 라이트 (50+ 개, 성능 주의 — 베이크 후 페이드)
- **Skybox**: 별+구름 일부. HDRI 1장 (직접 촬영 또는 CC0)

## 8. 사운드

- 앰비언스: 고공 바람 (VARCO Sound S06) + 저주파 도시 험
- BGM: `BGM_Salimter_Ambient`(~C6) → C6 후 `BGM_Ending_Theme` (서정 피아노 + 패드 + 한국 전통 현 악기 레이어)
- SFX: 바람 걸음, 잔디 소리, 파티클 쉼표
- Voice: C5, C6, 엔딩 내레이션 — VARCO Voice 최상 품질 트랙
