# Zone 1 · 알림터 (Alimter)

> DDP 진입 구역. 외벽 곡면과 광장의 경사로를 활용한 튜토리얼 존.
> 소요 시간 목표: **5~6분** · 난이도: ★~★★

## 1. 공간 레이아웃

```
     [ 외벽 램프 - 입구 ]
            │
            ▼
     ┌─────────────────┐
     │   광장 (Plaza)   │   ← 오프닝 스폰 / 첫 AI 라인
     └─────┬───────────┘
           │
           ▼
     [ 벽면 조형물 ]  ← C1 위치 (정면 크리스털)
           │
           ▼
     [ 곡면 외벽 통로 ] ← C2 위치 (이중 반사 퍼즐)
           │
           ▼
     [ Transition 1 → Z2 ]
```

면적 ≈ 40 m × 25 m. 최대 높이 8 m.

## 2. 스폰 & 오프닝

- 플레이어 스폰: 광장 서쪽 끝, DDP 외벽을 바라보는 각도
- 초기 시야: 주 조명 OFF, 비상 앰버 조명만 곡면 위에 점선처럼 흐름
- 오프닝 트리거 (5초 이내 자동 재생):
  - `"좋은 밤이에요, 율. 오늘 당직이군요."`
  - `"주 전원이 살아나질 않네요. 보조 광원만으로 알림터 크리스털 두 개를 다시 깨울 수 있어요."`
- 첫 노드 등장: 플레이어 전방 8 m 위치의 벽에 파란 맥동 링

## 3. 튜토리얼 퍼즐 P1.1 — "첫 반사" (C1)

### 목표
단일 Mirror 프리즘으로 직선 광원을 90° 굴절시켜 C1에 맞히기.

### 공간 배치
```
 광원 A  ─────►  [Node X]   ▼
                            ▼
                         [ C1 ]
```

### 튜토리얼 유도
- AI: `"손에 든 프리즘, 보이세요? 빛이 닿는 곳에 놓으세요. 빛이 알고 있는 길이 있습니다."`
- 노드 X에 시선 정렬 시 프리즘 미리보기 고스트 표시
- 설치 후 빔이 C1에 도달. 1초 유지로 활성화

### 실패 상태
- 잘못된 배치 시 빔이 벽에 충돌 → 조용한 "틱" SFX. 3초 동안 재시도 힌트 UI 없음
- 1분 이상 진전 없음 시 AI 힌트: `"빛은 정면에만 대답하지 않아요."`

### Vignette (C1 · 개관 2014, 22초)
[02 · Narrative §4.2](../02-narrative.md) 참조.

## 4. 심화 퍼즐 P1.2 — "곡면을 따라" (C2)

### 목표
외벽 곡면 경로에서 Mirror ×2로 **2회 반사** 후 C2 점등.

### 공간 배치
```
  광원 B  ─►  [Node Y1]
                 ▼
              (곡면 통로 위)
                 ▼
                [Node Y2]  ─►  [ C2 ]
```

### 설계 의도
- 플레이어가 DDP의 **곡면 동선**을 따라 걸으며 빔 경로를 구상하게 만든다
- 두 노드 사이에 직접 조준 불가능한 사각이 있어, 걸으며 뒤를 돌아볼 필요 있음

### 에지 케이스
- 플레이어가 Node Y2부터 배치하려 할 수 있음 → 광원이 Y1 쪽이라 작동 X. 의도된 교훈: "빛은 시작점이 정해져 있다"
- 노드 회전 각도 45° 단위 스냅이 있음. 최적해는 1개 조합만 크리스털 활성화

### Vignette (C2 · 첫 밤, 첫 불, 20초)
- 7세 아이 + 30대 어머니의 소리. 첫 개관 야간 이벤트 장면
- 마지막에 아카이브 AI의 말: `"… 나는 이 순간을 기억해요."` (첫 복선)

## 5. 환경 내러티브 (Environmental Storytelling)

- 광장 벤치 옆에 놓인 **식어 있는 커피 컵** (율의 동료가 놓고 간 암시)
- 벽면의 디지털 공지 패널 — 대정전 3주년 안내 (지문 수준으로만 읽힘)
- 외벽을 타고 흐르는 **오로라형 비상 조명** — 플레이어 이동 속도에 미세 반응

## 6. 라이팅

- **Key**: 없음 (주 조명 OFF 연출)
- **Fill**: 앰버 #FFB547, 강도 0.4, 외벽 곡면 emissive로 제공
- **Rim**: 쿨 화이트 #E8EEF5, 강도 0.2, 달빛 대체
- **Bake**: Zone 1 전체 베이크. 동적 라이트는 광원 A/B 2개만 Realtime

## 7. 사운드

- 앰비언스: 광장 실외 풍음 + 원거리 도로 소음 (VARCO Sound S01, S02)
- SFX: 프리즘 설치 "클릭"(SFX_prism_place), 빔 연결 "차임"(SFX_beam_connect)
- BGM: 없음 (이 존은 자연음만) — Zone 2에서 첫 BGM 진입
