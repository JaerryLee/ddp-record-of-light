# Zone 2 · 배움터 (Baeumter)

> DDP 실내 전시·아카이브 구역. Splitter + Filter 메커닉을 해금하고 색 퍼즐로 심화.
> 소요 시간 목표: **6~7분** · 난이도: ★★~★★★

## 1. 공간 레이아웃

```
   [ Z1→Z2 Corridor ]
          │
          ▼
   ┌──────────────────┐
   │  전시홀 (Hall)    │   ← Splitter 해금 지점
   └───┬──────────────┘
       │
       ▼
   [ 백색 회랑 ]       ← C3 Red 퍼즐
       │
       ▼
   [ 아카이브 랙 룸 ]  ← C4 Blue 퍼즐 + 첫 실마리
       │
       ▼
   [ 엘리베이터 홀 → Z3 ]
```

면적 ≈ 60 m × 35 m. 실내 천장 높이 7 m (아카이브 룸은 10 m 중공).

## 2. 해금 시퀀스 — Splitter 등장

플레이어가 전시홀 중앙에 진입하면:
- 천장 패널이 틸트하며 **흰 조명 기둥** 1개 하강
- AI: `"이곳의 빛은 한 가지 색만이 아니에요. 프리즘을 나누면, 빛도 나뉩니다."`
- 전시홀 구석 Node에 **Splitter 프리즘**이 물리적으로 놓여 있음 (플레이어가 상호작용으로 수령)

## 3. 퍼즐 P2.1 — "런웨이의 빨강" (C3)

### 목표
Splitter로 백색을 R/G/B 분할 → Filter(R)로 다른 색 차단 → C3에 적색 빔 도달.

### 공간 배치 (백색 회랑)
```
 광원 C  ─►  [Node S1]  (Splitter)
                  ├─ R 빔 ─►  [Node F1] (Filter R) ─►  [ C3 ]
                  ├─ G 빔 ─►  (통로 끝에서 흡수)
                  └─ B 빔 ─►  (통로 끝에서 흡수)
```

### 설계 포인트
- 플레이어가 Filter를 어느 빔에 놓아야 할지 직관으로 파악하도록 C3의 색을 **붉게 맥동**시킴
- 잘못된 필터 배치 시 C3 아예 반응 없음 (틱 사운드 없음 — 색 매칭 실패는 조용히)

### Vignette (C3 · 런웨이, 23초)
- 2018년경 서울패션위크. MC의 톤은 낮은 속삭임: `"... 처음 이 런웨이를 걸어요. 조명이 눈에 닿을 때, 사람들은 서로를 잊어요. 아주 잠깐만."`
- 홀로그램: 모델 실루엣 1명이 회랑을 걸어 지나감 (투명 50%, rim light 강)

## 4. 퍼즐 P2.2 — "큐레이터의 밤" (C4)

### 목표
아카이브 랙 룸. **두 개의 광원**에서 각각 다른 경로로 Blue 빔을 C4에 도달시키기. 두 빔이 동시 도달해야 활성화.

### 공간 배치
```
 광원 D ─► [Splitter] ─B─► [Filter B] ─► [ C4 ] ◄─B─ [Filter B] ◄─ [Splitter] ◄─ 광원 E
```

### 난이도 상승 요인
- 프리즘 동시 배치 수 제한 3개. 두 광원을 동시에 처리해야 하므로 경로 최적화 필요
- 아카이브 랙들이 시야를 가로막음 — 플레이어가 통로를 이동하며 구도를 재확인해야 함

### Vignette (C4 · 큐레이터의 밤, 24초)
- 심야 큐레이터의 모노로그: `"... 새벽 두 시. 전시 철거하러 왔는데, 이 작품 앞에 사람 하나가 있었어요. 울고 있더라고요. 물어보진 않았어요. 그냥 조명을 더 낮췄죠."`
- **마지막 4초에 잡음 섞인 음성**: `"유선희 박사에게 전해주세요."` (플롯 복선)

## 5. 환경 내러티브

- 아카이브 랙 중 일부엔 **물리 크리스털 샘플** 전시. 해독 불가한 홀로그램 명판
- 전시홀 천장엔 **중단된 패션쇼 포스터 현수막** — 먼지 쌓임
- 벽면에 "2042년 3월 23일" 날짜가 반복 출력된 디지털 시계

## 6. 라이팅

- **Key**: Emissive 조명 기둥 (백색, 강도 1.2). 씬 중심 조명
- **Fill**: 아카이브 랙에서 약한 사이언 라이트 (#6FDBFF, 강도 0.3)
- **Rim**: 사용 최소화 (실내라서)
- **Bake**: 전체 베이크. 조명 기둥만 Realtime (플레이어가 배경 경우)

## 7. 사운드

- 앰비언스: 실내 공조음 저주파 + 드물게 아카이브 랙의 하드드라이브 "딸깍" (VARCO Sound S03~S05)
- SFX: 색 필터 통과 시 색마다 고유 차임 (R=낮은 호른, G=현, B=벨)
- BGM: **첫 BGM 진입** (`BGM_Baeumter`). 미니멀 피아노 + 패드. 볼륨 -18 LUFS
